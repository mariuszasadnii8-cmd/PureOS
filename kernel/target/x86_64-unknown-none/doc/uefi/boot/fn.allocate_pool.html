pub fn allocate_pool(mem_ty: MemoryType, size: usize) -> Result<NonNull<u8>>

Allocates from a memory pool. The pointer will be 8-byte aligned.

§Errors

  • Status::OUT_OF_RESOURCES: allocation failed.
  • Status::INVALID_PARAMETER: mem_ty is MemoryType::PERSISTENT_MEMORY, MemoryType::UNACCEPTED, or in the range MemoryType::MAX..=0x6fff_ffff.
