Expand description

Enumerates all handles installed on the system which match a certain query.

§Errors

  • Status::NOT_FOUND: no matching handles found.
  • Status::BUFFER_TOO_SMALL: the buffer is not large enough. The required size (in number of handles, not bytes) will be returned in the error data.
