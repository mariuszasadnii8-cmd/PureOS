Expand description

Event type passed to Tcg::hash_log_extend_event.

Layout compatible with the C type EFI_TCG2_EVENT.

The TPM v1 spec uses a single generic event type for both creating a new event and reading an event from the log. The v2 spec splits this into two structs: EFI_TCG2_EVENT for creating events, and TCG_PCR_EVENT2 for reading events. To help clarify the usage, our API renames these types to PcrEventInputs and PcrEvent, respectively.
