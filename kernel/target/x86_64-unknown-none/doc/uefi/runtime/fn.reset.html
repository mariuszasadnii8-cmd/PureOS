pub fn reset(reset_type: ResetType, status: Status, data: Option<&[u8]>) -> !
Expand description

Resets the computer.

See ResetType for details of the various reset types.

For a normal reset the value of status should be Status::SUCCESS. Otherwise, an error code can be used.

The data arg is usually None. Otherwise, it must contain a UCS-2 null-terminated string followed by additional binary data. For ResetType::PLATFORM_SPECIFIC, the binary data must be a vendor-specific [Guid] that indicates the type of reset to perform.

This function never returns.
